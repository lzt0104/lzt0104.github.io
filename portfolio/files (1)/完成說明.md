## ⚠️ 重要提醒

1. **修改密碼**
   - wrangler.toml 第 18 行
   - blog-frontend.jsx 第 15 行
   - 兩處必須相同！

2. **設定 R2 公開訪問**
   - Cloudflare Dashboard → R2 → blog-images
   - Settings → Public Access → Enable
   - 複製 URL 到 wrangler.toml

3. **綁定資源**
   - Pages → Settings → Functions
   - 綁定 D1 (DB) 和 R2 (BLOG_IMAGES)
